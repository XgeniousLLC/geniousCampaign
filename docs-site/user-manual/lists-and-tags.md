## Lists & Tags

What this page is for: organizing contacts into named lists and tags so you can target them from campaigns, sequences, and triggers.

**Where to find it:** sidebar > **Lists & Tags** (`/lists`).

### Creating a list

1. In the **Lists** panel, type a name into the "New list name…" field and click **Add**.
2. The new list appears immediately with a member count of 0 — add contacts to it from the Contacts page (bulk **Add to list**), the CSV import mapping step, or a single contact's profile.
3. Click a list's name to open its scoped view — the same Contacts table, filtered to just that list's members.

### Creating a tag

1. In the **Tags** panel, pick a color with the color swatch, type a name, and click **Add**.
2. Tags show a live member count and appear as colored pills anywhere a contact is shown (contacts table, contact profile, campaign audience picker).

### Understanding list types

Every list has a **type** badge: **static** or **dynamic**.

- A static list is a fixed set of contacts you add explicitly (via CSV import, bulk action, or the contact profile).
- A dynamic list's badge is shown for context, but this page itself is where you create/view lists and their member counts — it doesn't currently expose a dynamic-list rule builder.

### Things to know

- Deleting a list or tag isn't available from this screen — it's create/view only here.
- Viewer-role users can see lists and tags but can't create new ones (the "New list name…"/"New tag…" inputs are hidden for them).
- A contact can belong to any number of lists and any number of tags simultaneously — they're independent organizational tools, not mutually exclusive categories.
